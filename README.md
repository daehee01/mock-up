# 위신싸 — B2C 랜딩 목업

벨루가 B2C(B2B2C) 확장 브랜드 **위신싸**("위스키가 신발보다 싸다")의 랜딩 목업.
인플루언서 전용 링크 → 상품 랜딩 → 스마트오더 픽업.

## 페이지
| 파일 | 역할 |
|---|---|
| `index.html` | 메인 — 지금 픽업 가능 캐러셀 + 프로모션 중 |
| `product.html` | 상품 — 인플루언서 링크(`?ref=`) 도착지 |
| `about.html` | 브랜드 소개 — 무엇인지·왜 싼지·픽업 방식 |
| `style.css` | 공통 디자인 시스템 (포레스트 그린 + 앰버, Pretendard) |

## 보기
`index.html`을 브라우저로 열면 됩니다. 모바일 퍼스트(max-width 480px)라 개발자도구 기기 모드 권장.

## 상태
정적 목업. 결제·어트리뷰션(`?ref=`)·매장 검색·성인인증은 `{{ }}` placeholder.
